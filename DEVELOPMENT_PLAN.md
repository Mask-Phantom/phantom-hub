# Phantom Hub - Development Roadmap

- [ ] Database Schema: Profiles (w/ tags), Rooms (Groups), Messages.
- [ ] Authentication: Email/Pass + MFA (Supabase).
- [ ] Security: Implement Row-Level Security (RLS) policies.

## Phase 2: Core Communication
- [ ] Real-time WebSocket Messaging (Private + Group).
- [ ] Group Management: Create/Join/Manage rooms.

## Phase 3: Social & Identity
- [ ] Project Feed: Showcase projects with identity tags.
- [ ] Profile Display: Role-based badges (e.g., "Frontend Engineer").

## Phase 4: Pro Features & Hardening
- [ ] Premium Gating: Implement "Incognito Mode" (Anonymity toggle).
- [ ] Security Hardening: XSS protection and Rate Limiting.
